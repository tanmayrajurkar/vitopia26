# Hybrid HLS – local segments (fast start)

The **manifest** (`index.m3u8`) is not stored here — it’s generated by `/api/hls/manifest` and served at `/hls/index.m3u8` via rewrite. Do not add a static `index.m3u8` to this folder, or segments 011–035 will be requested from localhost (404).

Put these 11 segment files in this folder so the hero video starts from your app (no Supabase round-trip for the first segments):

- `segment_000.ts`
- `segment_001.ts`
- `segment_002.ts`
- `segment_003.ts`
- `segment_004.ts`
- `segment_005.ts`
- `segment_006.ts`
- `segment_007.ts`
- `segment_008.ts`
- `segment_009.ts`
- `segment_010.ts`

Segments `segment_011.ts` through `segment_035.ts` are streamed from your Supabase **video** bucket.

**Supabase bucket:** Keep only `segment_011.ts`–`segment_035.ts` there. You can remove `index.m3u8` and `segment_000.ts`–`segment_010.ts` from Supabase (the app serves the manifest and first 11 segments).
